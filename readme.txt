Create custom skin for UAE's Posts widget using Elementor Page Builder for free.

## Description ##

You can create a custom post skin and asisgn it to the UAE's Posts widget.

= STEPS TO USE THE PLUGIN =

Step 1 - Visit Dashboard > Templates > Theme Builder.

Step 2 - Add New > Select'Post Skin' Type.

Step 4 - Edit it with Elementor and design a section.

Step 5 - Save it!

Step 5 - Go to your UAE's Post widget.

Step 5 - From skin type select custom skin.

Step 5 - Save it and you are done!
